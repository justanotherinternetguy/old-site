About me

Hey there! I'm justanotherinternetguy, a teenager learning to code in my free time. Since I was in elementary school, I have loved working with computers and messing around with code. My first venture in coding was fiddling with the source code of websites, changing colors and font sizes with the "Inspect Element" tool. I didn't really know what I was doing, only the fact that something was happening. From there, my love for computers was unfailing. I started learning how to actually code by the end of 4th grade, and I continued to hone my abilities ever since. Aside from computer science, I also enjoy Formula 1, Robotics, Biology, and Physics.

Hi! You've discovered the nerd corner. I have experience in multiple programming languages, frameworks, and tools (scroll down, after reading this of course). My current programming language is C++ due to its speed, powerful STL, and its popularity. However, my go-to language for small calculations and quick projects are still in Python because of how easy and quick it is to set up and get running. Python's minimalistic approach makes it easy and convenient to use. Like many others, I have a pretty decent amount of experience in Java (due to CS courses at school), but still dislike the language. It is verbose and way too object-oriented for my liking. I understand that OOP is an innovative new way to design programs, but I don't think it should be forced upon a programmer (like in Java). OOP should be a way to code, but not the only way. Also, I use vim!
